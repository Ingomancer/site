Vi reder också ut vem som är äldst av Andreas, Haskell, och Erlang.

Lite exempelkod:

sumAllNumbers :: String -> Int
sumAllNumbers str = sum (map read (lines str))

main :: IO ()
main = do
  fileContent <- readFile "magiska_tal.txt"
  print (sumAllNumbers fileContent)
